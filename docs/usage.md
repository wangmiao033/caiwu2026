# 使用说明

## 系统模块

- 导入：标准模板导入、手动录入、原表提取导入
- 核对任务：任务列表、确认账期
- 账单：生成账单、发送、试算预览
- 规则配置：游戏+渠道维度配置规则
- 发票/回款/财务看板

## 导入规则（关键）

- 导入阶段只导入原始数据：游戏、渠道、流水
- 不导入计算结果字段（业务毛利、结算金额、利润等）
- 规则字段统一在系统规则配置页维护

## 本地联调

1. 启动后端：`backend` 目录运行 `uvicorn app:app --reload --port 8000`
2. 启动前端：`web-admin` 目录运行 `npm run dev`
3. 登录后进入各页面操作
