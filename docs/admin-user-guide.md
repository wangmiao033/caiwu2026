# 管理员使用手册

## 1. 账号与权限
- 当前为简化单管理员模式，管理员账号由环境变量配置。
- 管理员可访问主数据、规则、审计日志及业务全流程页面。

## 2. 上线前检查
- 检查 `主数据` 是否完整（渠道、游戏、映射）。
- 检查 `规则配置` 是否覆盖核心渠道与游戏。
- 检查后端环境变量（数据库、JWT、管理员账号）。

## 3. 日常运维建议
- 每日查看 `首页待处理事项`。
- 每周查看 `审计日志` 是否存在异常操作。
- 导入前先做数据抽样校验，避免批量错误。

## 4. 审计日志使用
- 进入 `审计日志` 页面按条件筛选。
- 重点关注：删除动作、规则批量导入、账单发送、账期确认。

## 5. 数据备份建议
- 主数据与规则支持页面导出备份（Excel）。
- 数据库文件请按《数据库备份与恢复说明》定期备份。

## 6. 常见问题
- **管理员无法登录**：检查 `ADMIN_USERNAME/ADMIN_PASSWORD` 环境变量是否正确。
- **页面 401**：确认 token 是否过期，重新登录。
- **账单异常**：先检查核对任务是否已确认、规则是否生效。
